# lilToon Dissolve 效果

---

## 概述

Dissolve 用于材质过渡动画，如出现/消失、变身等效果。

---

## 参数详解

### 效果类型

| 参数 | 说明 |
|------|------|
| **Dissolve** | 效果类型 |
| **形状** | 效果形状 |
| **範囲** | 效果范围 |
| **ぼかし** | 边缘模糊量 |
| **マスク** | 透明度模式遮罩 (R通道) |
| **座標/向き** | UV/坐标模式的位置和方向 |
| **ノイズ** | 添加细节的遮罩 (R通道) |
| **色** | 效果颜色 |

---

## 形状类型

### UV 模式

| 形状 | 说明 |
|------|------|
| **UV** | 基于 UV 溶解 |
| **座標** | 基于世界/局部坐标 |

### 方向

控制溶解开始位置和移动方向。

---

## 噪声应用

### 用途

- 增加溶解边缘细节
- 创造不规则溶解效果
- 模拟烧毁/侵蚀

### 遮罩通道

- 噪声使用 R 通道

---

## 颜色效果

### 边缘颜色

Dissolve 边缘可添加颜色：
- 燃烧效果（橙/红色边缘）
- 能量效果（蓝/紫色边缘）
- 腐蚀效果（绿/灰色边缘）

---

## 典型应用

### 出现/消失动画

```
1. 设置 Dissolve 形状和范围
2. 动画控制时间/强度
3. 配合其他效果（发光等）
```

### 变身效果

```
1. 设置溶解形状
2. 添加边缘颜色
3. 控制溶解速度和方向
```

### 部分溶解

```
1. 使用遮罩控制溶解区域
2. 创建选择性溶解效果
3. 配合纹理细节
```

---

## 与其他系统配合

### 层级系统

Dissolve 可应用到 2nd/3rd 层级：
- 仅溶解特定图层
- 创造复杂过渡效果

### AudioLink 同步

配合 AudioLink：
- 音乐触发溶解
- 节奏同步动画

---

## 性能注意

> ⚠️ Dissolve 需要额外的着色器计算
> 避免在大量材质上同时使用
