# lilToon 渲染模式详解

---

## 渲染模式一览

| 渲染模式 | 负荷 | 说明 |
|---------|------|------|
| **不透明 (Opaque)** | ⚪ 最低 | 忽略透明度 |
| **カットアウト (Cutout)** | 🟢 低 | 透明/无半透明 |
| **半透明 (Transparent)** | 🟡 中 | 透明/半透明物体 |
| **屈折 (Refraction)** | 🟠 高 | 折射扭曲 |
| **屈折ぼかし (Refraction Blur)** | 🔴 高 | 折射+磨砂玻璃 |
| **ファー (Fur)** | 🔴 高 | 毛发生成 |
| **ファー(カットアウト) (Fur Cutout)** | 🟠 中高 | 毛发硬质 |
| **ファー(2パス) (Fur 2Pass)** | 🔴 高 | 毛发混合 |
| **宝石 (Gem)** | 🔴 高 | 宝石折射 |

---

## 不透明 (Opaque)

### 说明
完全忽略透明度信息，用于不需要透明处理的物体。

### 典型用途
- 皮肤（无透明需求时）
- 实体服装
- 不透明道具

### 性能
✅ **最优** — 无透明度处理开销

---

## カットアウト (Cutout)

### 说明
支持透明度但不支持半透明。透明度低于 Cutoff 值时完全透明。

### 典型用途
- 镂空材质
- 树叶
- 部分透明服装

### 参数
| 参数 | 说明 |
|------|------|
| **Cutoff** | 透明度阈值，低于此值完全透明 |

### 性能
🟢 **低** — 简单透明度计算

---

## 半透明 (Transparent)

### 说明
支持半透明效果。多个半透明物体重叠时可能产生渲染问题。

### 典型用途
- 头发
- 表情/眨眼
- 半透明配件

### 注意事项

⚠️ **渲染问题**：
- 半透明物体重叠时可能看不到后面的物体
- 解决方案：调整 Render Queue

⚠️ **Render Queue 调整**：
- 前景物体使用较高的 Render Queue
- 较大数值 = 之后绘制 = 显示在上层

### 性能
🟡 **中等** — 透明度混合计算

---

## 屈折 (Refraction)

### 说明
透过物体时产生扭曲效果。用于玻璃、水晶等。

### 典型用途
- 玻璃材质
- 水滴效果
- 水晶饰品

### 参数

| 参数 | 说明 |
|------|------|
| **强度** | 扭曲程度 |
| **フレネル** | 视角相关的扭曲变化 |
| **メインカラーから色を取得** | 用主色作为折射色 |
| **色** | 折射色的叠加 |

### 性能
🟠 **高** — 需要折射计算

---

## 屈折ぼかし (Refraction Blur)

### 说明
在折射基础上增加磨砂玻璃般的模糊效果。

### 典型用途
- 磨砂玻璃
- 雾化玻璃
- 毛玻璃效果

### 性能
🔴 **高** — 折射+模糊双重计算

---

## ファー (Fur)

### 说明
生成毛发的专用模式。支持重力、物理模拟。

### 典型用途
- 毛茸茸的动物
- 毛发装饰
- 草地效果

### 相关参数
详见 [毛发设置](fur.md)

### 性能
🔴 **高** — 多层几何生成

---

## ファー(カットアウト) (Fur Cutout)

### 说明
使用 Cutout 方式渲染毛发，视觉效果较硬。

### 特点
- 渲染较硬朗
- 在支持抗锯齿的环境下有所改善

### 性能
🟠 **中高** — 比全透明毛发低

---

## ファー(2パス) (Fur 2Pass)

### 说明
混合透明和 Cutout 毛发，各取所长。

### 优点
- 透明毛发的柔和感
- Cutout 毛发的清晰度

### 性能
🔴 **高** — 两层毛发混合

---

## 宝石 (Gem)

### 说明
专门为宝石设计的渲染模式，具有复杂的折射效果。

### 典型用途
- 宝石
- 水晶
- 钻石效果

### 相关参数
详见反射设置中的 [宝石设置](../reflection-settings.md#宝石设置)

### 性能
🔴 **高** — 复杂折射计算

---

## 渲染模式选择指南

```
是否需要透明度?
├── 否 → 不透明
└── 是
    ├── 是否需要半透明?
    │   ├── 否 → カットアウト
    │   └── 是
    │       ├── 毛发? → ファー系
    │       ├── 玻璃/宝石? → 屈折 / 宝石
    │       └── 其他 → 半透明
    └── 是否为毛发?
        └── 是 → ファー系
```

---

## 性能优先级

```
不透明 > カットアウト > 半透明 > 屈折 > ファー > 宝石
  (低)                                          (高)
```
