## 用途

References Domain 主要服务于:

1. **知识库维护** — 追踪哪些知识来自哪个源头
2. **冲突解决** — 当知识库与新版本官方文档冲突时,留档决策
3. **审计** — 二次审查时,快速定位"哪些知识可能过时"
