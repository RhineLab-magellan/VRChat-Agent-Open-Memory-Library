# lilToon 反射与光泽设置

---

## ノーマルマップ設定 (Normal Map)

### 概述
法线贴图影响阴影、光泽等效果，不直接改变外观。

### 影响范围

```
阴影 | 逆光 | 光泽 | MatCap | RimLight | 拉米 | AudioLink | 折射 | 宝石
```

### 参数

| 参数 | 说明 |
|------|------|
| **ノーマルマップ** | 法线贴图 + 强度滑块 |
| **マスク** | 法线应用遮罩 (R通道) |

### 支持数量
- 2 张法线贴图（主 + 2nd）
- 2nd 贴图支持遮罩

---

## 異方性反射 (Anisotropy)

### 概述
各向异性反射，适合头发、梳状金属等。

### 参数

| 参数 | 说明 |
|------|------|
| **ノーマルマップ** | 各向异性法线图 |
| **マスク** | 效果遮罩 (R通道) |
| **適用先** | 应用目标 |
| **タンジェント方向の幅** | 高光切线方向宽度 |
| **バイタンジェント方向の幅** | 高光副切线方向宽度 |
| **Offset** | 高光位置 |
| **ノイズの強度** | 高光噪声强度 |
| **強度** | 高光强度 |
| **ノイズ** | 高光位置噪声 (R通道) |

---

## 逆光ライト (Backlight)

### 概述
背光效果，与 RimLight 不同，接收阴影并跟随灯光。

### 参数

| 参数 | 说明 |
|------|------|
| **色** | 背光颜色 |
| **範囲** | 背光范围 |
| **ぼかし** | 背光模糊 |
| **指向性** | 灯光方向性 |
| **視線方向の影響度** | 视角方向影响 |
| **影を受け取る** | 接收其他物体阴影 |
| **裏面で無効化** | 背面禁用 |

## 光沢設定 (Reflection)

### 概述
光泽（镜面反射）控制，高光和真实反射。

### 参数

| 参数 | 说明 |
|------|------|
| **滑らかさ** | 表面光滑度 (R通道) |
| **金属度** | 金属感 (R通道) |
| **反射率** | 环境光反射率 |
| **色** | 反射颜色 |
| **光沢のタイプ** | 光泽类型 |
| **複数ライトから光沢を生成** | 多光源光泽 |
| **環境光の反射** | 环境光反射 |
| **合成モード** | 通常/叠加/屏幕/乘法 |

## マットキャップ設定 (MatCap)

### 概述
MatCap 是一种预绘制的光泽球体，视角相关。

### 特点

```
- 视角跟随（固定光泽）
- 无需光照计算
- 易于使用和控制
```

### 参数

| 参数 | 说明 |
|------|------|
| **マットキャップ** | MatCap 纹理 |
| **UV1を合成** | UV1 作为 MatCap UV 混合 |
| **Z軸回転キャンセル** | 取消 Z 轴旋转（VR 头部旋转） |
| **パース補正** | 透视校正 |
| **VR時の視差の強さ** | VR 视差强度 |
| **マスク** | MatCap 遮罩 (RGB) |
| **ライティングを適用** | 应用灯光亮度 |
| **影部分で無効化** | 阴影区禁用 |
| **裏面で無効化** | 背面禁用 |
| **合成モード** | 通常/叠加/屏幕/乘法 |
| **カスタムノーマルマップ** | MatCap 专用法线 |

### UV1 混合用途

典型应用：头发高光
```
- 正面 UV1 投影
- Y 轴混合
- 限制上下运动
```

## リムライト設定 (RimLight)

### 概述
轮廓光，模拟边缘光效果。

### 参数

| 参数 | 说明 |
|------|------|
| **色 / マスク** | 颜色 + 遮罩 |
| **範囲** | 光照范围 |
| **ぼかし** | 模糊量 |
| **リムライトの細さ** | 轮廓细度（对比度） |
| **ライティングを適用** | 应用灯光亮度 |
| **影部分で無効化** | 阴影区禁用 |
| **裏面で無効化** | 背面禁用 |
| **ライト方向の影響度** | 灯光方向影响 |
| **直接/間接光の幅** | 亮/暗区宽度 |
| **ノーマルマップ強度** | 法线影响 |
| **VR時の視差の強さ** | VR 视差强度 |

## ラメ設定 (Glitter)

### 概述
闪光效果，模拟亮片质感。

### 参数

| 参数 | 说明 |
|------|------|
| **UV Mode** | UV 选择（UV1 优先） |
| **色** | 闪光颜色（HDR） |
| **メインカラーの強度** | 主色叠加强度 |
| **ライティングを適用** | 应用灯光 |
| **影/裏面で無効化** | 阴影/背面禁用 |
| **Shapeマスク** | 形状遮罩 (RGBA) |
| **アトラス** | 遮罩纹理图集数 |
| **ランダム化** | 随机旋转/大小 |
| **サイズ** | 闪光粗细 |
| **パーティクルサイズ** | 粒子大小 |
| **密度** | 闪光密度 |
| **感度** | 角度点灭灵敏度 |
| **点滅の速度** | 点灭速度 |
| **角度制限** | 角度限制 |
| **ライト方向の影響度** | 灯光方向影响 |
| **ランダムカラー** | 随机颜色 |
| **VR時の視差の強さ** | VR 视差 |
| **ノーマルマップ強度** | 法线影响 |
| **コントラスト（後処理）** | 后处理对比度 |

| **フレネル** | 视角折射变化 |
| **色収差** | 色差强度 |
| **コントラスト** | 环境光对比度 |
| **環境光の色** | 环境光叠加色 |
| **パーティクルループ数** | 粒子细度 |
| **色** | 粒子颜色 |
| **VR時の視差の強さ** | VR 视差 |
| **滑らかさ** | 表面光滑度 |
| **反射率** | 环境光反射率 |
