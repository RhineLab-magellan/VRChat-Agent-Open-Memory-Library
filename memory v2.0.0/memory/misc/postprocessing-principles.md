# 后处理原理 — 现实参照

> 来源: VRCD 文档库（docs.vrcd.org.cn）| 更新: 2026-06-05

---

## 后处理的两大目的

| 目的 | 说明 |
|------|------|
| **赋予真实感** | 模拟现实镜头/眼球的光学缺陷（散射、畸变、光晕等） |
| **为画面调味** | 手动调整明暗、色彩平衡、曲线，类似 P 图 |

---

## Motion Blur（动态模糊）

### 现实原理

曝光过程中，拍摄场景中物体发生**相对运动**（相机观测系下的相对运动）→ 光线改变 → 拍摄物体模糊。

人眼视觉暂留同样依赖光化学反应，**缺少动态模糊的画面反而会丧失运动感，使观众感觉断断续续不自然**。

### 数学公式

```
g(x,y) = ∫(0,T) f(x-s(τ), y-s(τ)) dτ × 曝光系数s(0-1)

- g(x,y): 运动模糊后在坐标(x,y)的结果
- T: 曝光时间
- s: 曝光系数，范围(0,1)
```

> 曝光系数用于调整，防止像素灰度级超出 0-255 范围。

## Bloom（炫光）

### 物理来源（多机制叠加）

| 来源 | 说明 |
|------|------|
| **镜头镀膜缺陷** | 内部产生寄生照明，漫反射光晕出现在光源周围 |
| **胶片乳剂不均匀** | 卤化银晶粒簇导致散射，老镜头尤为显著 |
| **艾里斑（Airy Disk）** | 点光源通过圆形孔径产生衍射，尾部无限宽 |
| **CCD 电荷溢出** | 高亮区域电荷溢到相邻像素（电荷出血） |

### 艾里斑原理

> 现实中的镜头永远无法实现完美聚焦。

- 艾里斑函数具有快速衰减特性，但尾部扩散范围极宽（理论上无限宽）
- 当图像相邻区域亮度相近时，模糊效应不明显
- 在明暗交界处，艾里斑尾部变得可见，影响范围可远超原亮部区域

**室内示例**：窗外亮度可能是室内物体的 70-80 倍，按室内曝光时窗外高亮区域的光晕会溢出窗框。

### 计算机实现思路

```
1. 根据阈值提取图像中的较亮区域 → 存储到渲染纹理
2. 高斯模糊模拟光线扩散效果
3. 与原图混合
```

### 前置知识

| 概念 | 说明 |
|------|------|
| **LDR（低动态范围）** | JPG/PNG，RGB 范围 [0,1]，无法表达真实世界色彩 |
| **HDR（高动态范围）** | HDR/EXR，RGB 可超出 [0,1]，可提取更亮区域产生精准 Bloom |
| **高斯模糊** | 减少图像噪声，降低细节层次 |

### 高斯模糊公式

```
权重 w(i,j) = (1/(2πσ²)) × e^(-(i²+j²)/(2σ²))
```

归一化作用：保证卷积后的图像不会变暗。

## Vignette（晕影）

### 三种类型对比

| 类型 | 成因 | 特征 |
|------|------|------|
| **自然晕影** | 光线传输衰减（余弦四次方规律） | 逐渐变暗，焦距越大/光圈越小越明显 |
| **光学晕影** | 光圈边缘遮挡光线（镜筒内部） | 大光圈最明显，缩光圈改善 |
| **机械晕影** | 遮光罩/滤镜物理遮挡 | 突然发生，只在角落 |

### 数学模型

场景辐照度与入射传感器能量关系：**余弦四次方规律**

- 离轴角度 α 越大，衰减越多
- 焦距 f 越大，衰减越多
- 光圈直径 d 越小，衰减越多
- 光圈值 N = f/d，光圈值越大（即光圈越小）衰减越多

### 机械晕影实例

- 尼克尔 16-35mm f/4 @ 16mm 即使无滤镜也有晕影
- 加圆偏振镜晕影加剧
- Lee 滤镜支架系统在 24mm 处表现糟糕
- 使用多个滤镜或滤镜支架系统会加剧晕影

## Grain（颗粒）

### 胶片颗粒 vs 数码噪点

| 类型 | 成因 | 视觉特征 |
|------|------|----------|
| **胶片颗粒** | 卤化银晶体在感光乳剂中的自然分布 | 随机光学纹理，无颜色（单色） |
| **数码噪点** | 传感器电子噪声 + 图像处理器缺陷 | 有颜色，降低影像品质 |

### 胶片颗粒特性

- 颗粒密度与 ISO 感光度直接相关：ISO 越高，颗粒越显著
- 高 ISO 胶片（800+）的颗粒效果反而备受青睐
- 赋予影像独特的质感魅力，电影摄影中用于营造怀旧氛围

### 数码噪点特性

- 随机像素失真，随亮度提升（暗光环境更明显）
- ISO 越高，噪点越多
- 像素较小的相机更明显（电荷消散空间小）

## Distortion（镜头畸变）

### 三种类型

| 类型 | 英文名 | 特征 |
|------|--------|------|
| **枕形畸变** | Pincushion Distortion | 画面向中间「收缩」，长焦镜头常见 |
| **桶形畸变** | Barrel Distortion | 画面呈桶形膨胀，广角镜头常见 |
| **线性畸变** | Linear Distortion | 近大远小的透视失真（梯形失真） |

### 关键点

- 镜头畸变由镜片放大率差异导致，是透镜固有特性
- 大多镜头会通过镜片组优化设计降低畸变，但无法完全消除
- 大多相机自动追加后期畸变矫正

### VR 注意事项

> ⚠️ **不建议在 VR 中添加畸变后处理**
>
> - 头显设备本身有厂商畸变校正
> - 人眼在大脑矫正下感知到的图像没有畸变
> - 此后处理仅适用于 Unity 摄影等项目
