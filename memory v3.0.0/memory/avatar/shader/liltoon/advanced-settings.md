# lilToon 扩展功能设置

---

## 輪郭線設定 (Outline Settings)

### 概述
线画效果，支持纹理控制粗细和颜色。

### 主要参数

| 参数 | 说明 |
|------|------|
| **テクスチャ** | 轮廓线纹理/颜色 |
| **色相/彩度/明度/ガンマ** | 颜色调整 |
| **焼き込み** | 导出校正后纹理 |
| **ハイライト** | 轮廓高光设置 |
| **ライティングを適用** | 应用光照 |
| **マスクと太さ** | 轮廓粗细遮罩 (R通道) |
| **太さを補正** | 距离粗细校正 |
| **頂点カラー** | 顶点颜色：None / R→Width / RGBA→Normal&Width |
| **太さ0の頂点を削除** | 删除零粗细顶点 |
| **Z Bias** | 前后移动轮廓 |
| **ノーマルマップ** | 轮廓挤出方向调整 |

### 顶点颜色烘焙

用于硬边模型的光滑轮廓：

```
步骤：顶点颜色烘焙工具
1. 准备光滑法线模型
2. 应用烘焙工具
3. 分配给轮廓线着色器
```

## 視差マップ (Parallax)

### 概述
视线方向 UV 偏移，模拟立体感。

### 参数

| 参数 | 说明 |
|------|------|
| **視差マップ** | 视差图 + 强度 (R通道) |
| **オフセット** | 基准值 |

---

## 距離フェード (Distance Fade)

### 概述
距离渐变效果，可用于远景或临场感。

### 参数

| 参数 | 说明 |
|------|------|
| **色** | 渐变后颜色 |
| **開始距離** | 渐变开始距离 |
| **終了距離** | 渐变结束距离 |
| **強度** | 渐变强度 |

### 用途示例

- 靠近时变暗（临场感）
- 距离渐变（雾效）
- 反向使用（远景消失）

## AudioLink

### 概述
VRChat 音频同步动画支持。

### 参数

| 参数 | 说明 |
|------|------|
| **UV Mode** | 采样 UV 类型 |
| **サイズ/Offset/角度** | UV 控制 |
| **帯域** | 采样频段 |
| **マスク** | RGB=延迟/频段/强度 |
| **AudioLink無効時の初期値** | 非対応世界行为 |
| **適用先** | 效果应用目标 |
| **ローカル化** | 使用本地纹理 |
| **ローカルマップ** | 本地纹理 (R通道) |
| **BPM/拍数** | 节奏同步 |
| **Offset** | 同步偏移 |

## Dissolve (溶解)

### 概述
溶解/消失效果，用于转场或变形。

### 参数

| 参数 | 说明 |
|------|------|
| **Dissolve** | 效果类型 |
| **形状** | 效果形状 |
| **範囲** | 效果范围 |
| **ぼかし** | 模糊量 |
| **マスク** | 遮罩纹理 (R通道) |
| **座標/向き** | UV/坐标模式位置和方向 |
| **ノイズ** | 细节遮罩 |
| **色** | 效果颜色 |

## レンダリング設定 (Rendering Settings)

### 概述
高级渲染参数，通常无需修改。

### 参数

| 参数 | 说明 |
|------|------|
| **シェーダーの種類** | 通常版/轻量版 |
| **Cull Mode** | 描画面 |
| **ZWrite** | 深度写入 |
| **ZTest** | 深度测试 |
| **Offset Factor/Units** | 深度偏移 |
| **SrcBlend/DstBlend** | 混合系数 |
| **BlendOp** | 混合操作 |
| **Enable GPU Instancing** | GPU 实例化 |
| **Render Queue** | 渲染队列 |

### Blend Mode 组合

| 模式 | SrcBlend | DstBlend |
|------|----------|----------|
| 通常透明 | One | OneMinusSrcAlpha |
| 叠加 | One | One |
| 屏幕 | OneMinusDestColor | One |
| 乘法 | DstColor | Zero |

## テッセレーション (Tessellation)

### 概述
靠近时平滑模型，仅用于影像制作。

### 参数

| 参数 | 说明 |
|------|------|
| **閾値** | 边分割阈值 |
| **強度** | 平滑强度 |
| **縮小補正** | 缩小校正 |
| **最大分割数** | 最大边分割数 |

> ⚠️ 负荷高，仅建议影像制作使用。

---

## 屈折設定 (Refraction)

### 概述
折射效果，用于玻璃、水晶等。

### 参数

| 参数 | 说明 |
|------|------|
| **強度** | 扭曲强度 |
| **フレネル** | 视角扭曲变化 |
| **メインカラーから色を取得** | 用主色作为折射色 |
| **色** | 折射色叠加 |

---

## ライティング・明るさ設定

### 概述
亮度控制参数。

### 参数

| 参数 | 说明 |
|------|------|
| **明るさの下限** | 暗部限制 (VRChat 建议 0.0-0.2) |
| **明るさの上限** | 亮部限制 |
| **ライトのモノクロ化** | 灯光饱和度 |
| **影色への環境光影響度** | 环境光对阴影影响 |
| **Unlit化** | 无光照模式 |
| **頂点ライトの強度** | 顶点光强度 |
| **ライト方向のオーバーライド** | 灯光方向覆盖 |
| **オブジェクトの向きに追従** | 跟随物体方向 |
| **[BRP] ライトの合成** | 追加灯光合成方式 |
| **[HDRP] 曝光限制** | 曝光前亮度限制 |
| **[HDRP] Directional Light 强度** | 方向光强度 |
